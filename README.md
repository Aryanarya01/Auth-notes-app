# 📝 Full Stack Notes App

A simple full stack notes application with authentication where users can create, edit, and delete their personal notes.

## 🚀 Tech Stack

Frontend
- React
- TypeScript
- Context API

Backend
- Node.js
- Express
- TypeScript
- MongoDB
- JWT Authentication

## ✨ Features

- User Register & Login
- JWT Cookie Authentication
- Protected Routes
- Create Notes
- Edit Notes
- Delete Notes
- User Profile

## ⚙️ Run Locally

Backend

cd backend
npm install
npm run dev

Frontend

cd frontend
npm install
npm run dev
